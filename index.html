HexClock

Single page web app with customizable display. Background color is based on percentage of day using HSL color values.

Customization options include speed of hue and saturation changes, font, what character string is used between the displayed data, and an array of information, including the real time to the second, and the currently displayed color.

Moon Phase widget

Uses the Astronomy API moon phase studio endpoint to get the image of the moon from a given location at a given time, and the US Navy Astonomical Applicationas Department Celestial Navigation Data for the phase name and percent illumination, again using the user's real time. Defaults to Rochester, NY as the location.

Uses styling to "flip" to show information, with a hover effect for dynamic interaction.

Connectunes

Music themed mini games using user's Spotify data. Created for WicHacks 2026, where we won 1st place for "Best Game Hack".

Uses Spotify API and lrclib API to pull user listening data and lyrics for their top songs.

Games include a NYT Connections inspired game grouping lyrics, a color by number nonogram using album art, and ranking songs by number of listens in the last month.

Minesweeper

Minesweeper game coded in Javascript, based on minesweeperonline.com with additional right click support rather than middle mouse or space bar. I coded this over the course of a week during my lunch breaks while working and it was a great test of my ability to refactor and design quickly. It also taught me about a lot of interactive javascript controls I had previously never used.

Gaia

Star mapping mini game prototype in figma. Created for WicHacks 2024, won 1st place in "Widget's Wonders" category for small passion projects.

This version of the project is a concept for a mobile game where players can either explore a star map, drawing their own constellations with the stars visible, or can play in a timed mode, given a star map from a random location at a random time, and try to find a given constellation within view.

Torrid Glen

Based on Toroko Gorge, a digital poem by Nick Montfort.

In this poem, lines are generated using word banks and a structured pattern for each line to create stanzas. My version is meant to evoke the motion of falling water, and uses words related to that natural environment.

This was made for a class assignment for Text and Code. Inspired by Watkins Glen, the state park in New York's Fingerlakes region.

Sunnarized

Multipage fictional news site covering brief research on how the COVID-19 pandemic and lockdowns affected education and remote work options. Created for a class final project to show comptency in research, HTML and CSS design, with both desktop and mobile display and responsive design.
