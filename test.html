Moon Phase widget

Uses the Astronomy API moon phase studio endpoint to get the image of the moon from a given location at a given time, and the US Navy Astonomical Applications Department Celestial Navigation Data for the phase name and percent illumination, again using the user's time and location. Defaults to Rochester, NY as the location if errors occur or the permission is denied.

Uses styling to "flip" to show information, with a hover effect for dynamic interaction.

HexClock

Single page web app with customizable display. Background color is based on percentage of day using HSL color values.

Customization options include speed of hue and saturation changes, font, what character string is used between the displayed data, and an array of information, including the real time to the second, and the currently displayed color.

Connectunes

Music themed mini games using user's Spotify data. Created for WicHacks 2026.

Uses Spotify API and lrclib API to pull user listening data and lyrics for their top songs.

Games include a NYT Connections inspired game grouping lyrics, a color by number nonogram using album art, and ranking songs by number of listens in the last month.
